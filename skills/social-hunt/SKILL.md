# Social Media Hunt Skill

Search for a person across social media and online platforms:

1. **Username Search** — Use Sherlock to check 400+ platforms
2. **Profile Analysis** — Gather public profile data from found accounts
3. **Cross-reference** — Link accounts that likely belong to the same person
4. **Activity Analysis** — Note active vs inactive accounts
5. **Metadata** — Extract any useful metadata from profiles

Present findings as a list of confirmed accounts with URLs.
Note confidence level for each match (high/medium/low).
